Cursor Orb

Red-glow orb follows cursor with linear interpolation (lerp 0.12). Pointer-events none para não bloquear hovers. Desligado em prefers-reduced-motion.

Grid Parallax

Grade cartesiana se desloca no sentido oposto ao cursor com intensidade 0.03. Sensação orgânica de profundidade sem tirar foco do conteúdo. Eixos vermelhos permanecem fixos.

Drawer Transitions

V1 Right Drawer dockado no desktop; abrir/fechar com spring cubic-bezier(0.25, 1, 0.5, 1). Aparece instantâneo nesse protótipo — em produção o AnimatePresence do Framer Motion anima de translateX(100%).

Command Palette (V3)

Trigger via FAB no mobile (equivalente de ⌘K no desktop). Entrada: fade-in 200ms + scale 0.94→1 260ms. Primeiro item sempre destacado (prontidão pra tecla Enter).

FAB Pulse

Pulso de glow a cada 2.4s pra sinalizar presença sem ser invasivo. Suspenso em reduced-motion.

Typography Stack

Display: Fraunces (italic nos lastnames). Body: IBM Plex Sans. Mono: JetBrains Mono (kickers, badges, mensagens do Axis).

Diagonal Tilt (d8 inheritance, amplificado)

Grid rotaciona em torno de BASE_TILT = 5° com ajuste baseado em cursor: tilt = 5 + px * 2.7 − py * 2.0. Range efetivo ~2.6° a ~7.4° (d8 original era 1.4°–4.6°). Suficiente pra sentir o movimento em descanso sem cair em vertigem.

Coord Plot Label

Ao lado do orb, (x, y) em unidades cartesianas (unit = 12px). Origem no centro do viewport desktop; y invertido pra seguir matemática. Desaparece se o cursor sair do viewport.

Easter Egg Hotspots

5 pontos plotados em coords conhecidas — (0,0) origem, projetos históricos, stack atual. Herdam tilt + parallax do grid (ficam "ancorados ao universo"). Hover revela tooltip com contexto.

Origin Pulse + Orbit (d8 inheritance)

Dois anéis concêntricos em (0,0) pulsam em periodos dessincronizados (2.6s + 3.8s) — respiração orgânica. Um grupo orbita o centro a cada 16s; o dot traça um círculo dashed. Reforça "origem viva" em vez de ponto estático.

Dashed Trajectory + Leading Dot

Linha stroke-dasharray: 3 6 conecta os eggs na ordem origem → Snake → Portfolio → TSE → JPG Labs → origem (fechamento). Leading dot percorre a trajetória em 11s com animateMotion rotate="auto". Visual evoca "cronologia" do portfólio.

Motion handoff · d9 → production

Smart Animate × target code stack

Mapeamento explícito de cada motion (M1–M11) pro stack de consumo. Onde o Smart Animate não cobre (cursor-driven, rAF, SVG animateMotion, scroll-driven complexo), o Figma entra só como referência estática e o motion nasce direto em código.

# Motion Smart Animate? Target stack Notes
M1 Cursor Orb
(lerp follow)		 Não cobre rAF loop + pointermove Interpolação contínua (lerp 0.12) depende de event loop. Figma não tem cursor-follow nativo — documentar só estado inicial/repouso.
M2 Grid Parallax Não cobre rAF + CSS transform ou GSAP quickTo Deslocamento −0.03 oposto ao cursor. Em React: useMotionValue + useSpring (Motion). Em produção, Lenis + GSAP para smooth scroll combinado.
M3 Diagonal Tilt Não cobre rAF + CSS rotate() Fórmula BASE_TILT + px·TILT_X − py·TILT_Y. DS default (3° / 1.6 / 1.2) para produto; showcase usa 5° / 2.7 / 2.0. Respeitar prefers-reduced-motion.
M4 Coord Plot Label Não cobre rAF + Math.round (UNIT=12px) Tooltip (x, y) recalculado por frame. Fade-in via opacity + var(--ease-out-quart) 200ms — essa parte cabe em Smart Animate mas o conteúdo do label é dinâmico.
M5 Easter Eggs
(pulse + ring)		 Cobre (loop) CSS @keyframes infinite Showcase-only. Não shippar em Axis/jpglabs.com.br. Se reaproveitar o pulse ring em outro contexto, OK — é keyframes simples sem JS.
M6 Origin Pulse Rings Cobre (loop) SVG <animate> ou CSS @keyframes Dois anéis dessincronizados (2.6s + 3.8s). SVG animate é mais fiel ao protótipo. Em React, motion.circle com animate={...} + repeat: Infinity.
M7 Orbit Ring Cobre parcial SVG animateTransform ou GSAP timeline Rotação contínua em torno do origin. Figma anima via After Delay em loop curto, mas dashed stroke + velocidade constante fica mais limpo em SVG ou gsap.to(..., { rotation: 360, repeat: -1 }).
M8 Trajectory + Leading Dot Não cobre SVG animateMotion + <mpath> Dot percorrendo path dashed em 11s com rotate="auto". Em React/GSAP: gsap.to(dot, { motionPath: { path, autoRotate: true } }) (plugin MotionPath).
M9 Drawer V1 (open/close) Cobre Framer Motion AnimatePresence Translate translateX(100% → 0) 300ms var(--ease-out-quart). Smart Animate entre 2 frames (drawer fechado/aberto) replica direto. Cuidar de focus trap e aria-modal no código.
M10 Mobile Palette V3
(⌘K)		 Cobre CSS @keyframes ou Framer Motion Overlay fade-in 200ms + palette scale 0.94 → 1 260ms. Smart Animate entre 2 frames (fechado/aberto). Trigger via FAB (mobile) e ⌘K (desktop). Primeiro item pré-destacado.
M11 FAB Pulse Cobre (loop) CSS @keyframes 2.4s Box-shadow oscilando a cada 2.4s. Suspenso em prefers-reduced-motion. Baixo custo — pode ficar puro CSS em produto.
— Theme swap (dark ↔ light) Cobre CSS custom props + data-theme Swap instantâneo dos tokens (sem transição). DS contract: polaridade flipa inteira de uma vez, accent (#c04a3f) permanece estável. Persistir em localStorage; honrar prefers-color-scheme no primeiro load.
